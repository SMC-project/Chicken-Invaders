----------------Main Menu Controls
-in main menu you can start the game by pressing 'space' and select the options on the left with 'z' / 'enter'
-in the leaderboard you can type your name at the end of the game; press 'enter' to save the data and compete with your friends
-press 'esc' to exit the leaderboard

-----------------In-game Controls
use the arrows to move
'z' to fire
'x' to shoot a rocket
'esc' to pause the game
